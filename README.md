Bu projemde Image Processing (Görüntü İşleme) alanında kullanılan makine öğrenmesi tabanlı yöntemlerin matematiksel altyapısını teorik açıklamalar ve Python uygulamaları ile ele alır.
Amaç; görüntü işleme algoritmalarının arkasındaki lineer cebir, kalkülüs ve olasılık temellerini net bir şekilde kavramaktır.
